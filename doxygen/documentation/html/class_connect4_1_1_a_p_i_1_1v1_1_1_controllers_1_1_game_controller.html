Detailed Description

Controller for managing game-related actions.

Provides endpoints to create, join, leave, and play moves in games.

◆ CreateGame()

async Task< IActionResult > Connect4.API.v1.Controllers.GameController.CreateGame ( [FromBody] string gameName)

Creates a new game.

This endpoint allows an authenticated user to create a new game by providing a valid game name.

Possible HTTP status codes:

  • 200 OK - Game created successfully.
  • 401 Unauthorized - The user is not authenticated.
  • 400 Bad Request - Invalid game name (e.g., empty, too short, too long).
  • 409 Conflict - Conflict in creating the game (e.g., user is already in a game).
  • 500 Internal Server Error - Unexpected internal error.
Parameters
gameNameThe name of the game to create.
Returns
Returns a JSON object containing the GameIdResponsDto if the game is created successfully. In case of an error, a JSON object containing the MessageResponsDto is returned.
