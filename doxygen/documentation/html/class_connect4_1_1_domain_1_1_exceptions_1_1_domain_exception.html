Detailed Description

Represents a generic domain exception where the default behavior is to treat it as a client-side error (400 Bad Request).

This exception is used to indicate errors that occur due to invalid operations or inputs within the domain logic. It is typically treated as a client-side error, resulting in a 400 Bad Request HTTP response.
