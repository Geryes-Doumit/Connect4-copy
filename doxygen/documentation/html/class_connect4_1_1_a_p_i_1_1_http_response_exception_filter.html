Detailed Description

Global exception filter that transforms custom domain exceptions into appropriate HTTP status codes.

  • UnauthorizedDomainException => 401 (Unauthorized).
  • BusyUserDomainException => 409 (Conflict).
  • GameNotFoundException => 404 (Not Found).
  • NotYourTurnDomainException => 403 (Forbidden).
  • DomainException => 400 (Bad Request).
  • Otherwise => 500 (Internal Server Error).
