Detailed Description

Controller that handles the list of games (waiting and finished).

Provides endpoints to list all games waiting for a player and to retrieve the history of games where the requester was either the host or the guest.

async Task< OkObjectResult > Connect4.API.v1.Controllers.GamesListController.GetFinishedGames ( [FromQuery] int? limit,
[FromQuery] int? offset )

Lists all games with status "Finished" where the current user is host or guest.

This endpoint retrieves all finished games in which the authenticated user participated either as the host or as the guest. It supports pagination through optional limit and offset query parameters.

Possible HTTP status codes:

  • 200 OK - Successfully retrieved the list of finished games.
  • 401 Unauthorized - The user is not authenticated.
  • 400 Bad Request - Invalid query parameters.
  • 409 Conflict - Conflict in retrieving the games (e.g., user is currently in a game).
  • 500 Internal Server Error - Unexpected internal error.
Parameters
limitOptional. The maximum number of games to return. Defaults to 10 if not specified.
offsetOptional. The number of games to skip before starting to collect the result set. Defaults to 0 if not specified.
Returns
Returns a JSON array of <see cref="FinishedGameDto"

objects representing the finished games." In case of an error, a <see cref="MessageResponsDto"/> object is returned.

Lists all games whose status is "WaitingForPlayers".

This endpoint retrieves all games that are currently waiting for players to join. It supports pagination through optional limit and offset query parameters.

Possible HTTP status codes:

  • 200 OK - Successfully retrieved the list of waiting games.
  • 401 Unauthorized - The user is not authenticated.
  • 400 Bad Request - Invalid query parameters.
  • 409 Conflict - Conflict in retrieving the games (e.g., user is currently in a game).
  • 500 Internal Server Error - Unexpected internal error.
Parameters
limitOptional. The maximum number of games to return. Defaults to 10 if not specified.
offsetOptional. The number of games to skip before starting to collect the result set. Defaults to 0 if not specified.
Returns
Returns a JSON array of <see cref="WaitingGameDto"

objects representing the waiting games. In case of an error, a MessageResponsDto object is returned.
